1. 3D 視覺展廳與透視投影
3D 紙盒 (Box) 與手提袋 (Bag) 渲染：支援雙包裝類型 3D 實體模型無縫紋理貼圖。
真實空間透視圖 (PerspectiveCamera)：相機採用真實透視投影（fov: 45），具備真實的「近大遠小」空間感與立體高質感。
固定視角管理系統：
內建 7 大常用固定視角（正面、背面、左側、右側、上方、下方、45 度斜角）平滑自動跳轉。
自訂視角新增與刪除：調整好視角後可一鍵「儲存目前視角」，支援重新命名與刪除二次確認。
localStorage 永久儲存：視角以專案檔案名稱進行隔離，永久儲存於瀏覽器，下次打開自動恢復。
相機雙向即時微調控制：
實時數值展示：滑鼠旋轉或縮放時，即時於 UI 上更新相機的位置 (X, Y, Z)、焦點與 Zoom 數據。
雙向手動微調：可直接在 UI 上輸入數值微調 3D 視角，帶有智慧「打字輸入防抖鎖定」，避免無限重繪死鎖。
2. 2D 智慧網格對齊與雙引擎 AI 定位
雙引擎 AI 自動折線定位：
可自由選用 Gemini 3.5 Flash（啟用高規格 MEDIA_RESOLUTION_HIGH 傳輸）或 GPT-5.5（max_completion_tokens: 16000 配合極速推理抑制指令），高精度自動辨識 dieline 折線與長寬比。
所有座標數據（x, y, w, h）均精確計算並四捨五入至小數點後 4 位。
刀模折線緊密貼合與黑框上限截斷：
智慧裁切線緊貼「啡紅色 fold 折線」進行定位，若遇到「黑色 outer 裁切線」則自動以此為終點進行截斷，徹底裁除外圍多餘的紙張白邊。
手動拖曳座標微調：
智慧對齊網格上設有可視化的垂直與水平對齊線，支援手動拉動線條，實時更新裁剪區域。
2D 面拖曳對調 (Drag & Drop) 與 3D 同步：
在 2D 網格的面標籤（Back, Left, Front, Right）上支援滑鼠原生拖曳對調。
拖曳對調放開後，即時更變面順序狀態，並即時通知 3D 視窗同步更換貼圖材質。
3. 獨立截面裁切與 Three.js 貼圖中心旋轉
2D 獨立截面預覽：裁切出的面（包含頂蓋與底面）均有獨立的預覽小圖。
2D 🔄 旋轉 90° (微動畫)：點擊旋轉按鈕，2D 預覽圖套用 CSS transition 平滑轉動。
3D 材質正中心軸旋轉：在 Three.js 載入貼圖時套用中心旋轉 texture.center.set(0.5, 0.5)，貼圖旋轉 90° 時完美固定於面板正中央，無任何拉伸形變。
4. 智慧行銷合成工具 (端到端直接生成場景)
移去手動去背與 Canvas 合成：直接導入原始商品照或 2D 裁剪圖，100% 完整保留商品形狀、LOGO、標籤與文字細節，避免去背失真。
雙階段多模型一體化場景生成：
理解階段：在 UI 自由切換 gemini-3.5-flash、gemini-3.1-pro 或是 gpt-5.5 多模態理解商品特徵並產生融合 Prompt。
生成階段：由 gemini-3.1-flash-image-preview 或是 gpt-image-2-2026-04-21 一體化生成大圖，光影與物理高光完美融合，無生硬「貼紙感」。
動態 API 雙金鑰輸入支援：依據所選模型，智慧且動態切換顯示「Google AI Studio API Key」或「OpenAI API Key」金鑰欄位。
5. 一鍵匯出全套與「啡紅邊緣智慧剔除」
啡紅邊界像素掃描裁切：
點擊「一鍵匯出全套」前，演算法利用 Canvas ImageData 對要導出的圖片進行四周邊緣像素掃描，精確辨識偏啡紅色的刀模線並自動向內收縮邊界。
15% 最大收縮上限保護：防禦性防止包裝本身的紅色設計被誤切。
100% 純淨等比置中導出 (1000x1000)：
剔除紅邊後，重新將主體產品等比縮放（較長投影邊限制在 800px 內），置中繪製於 1000 * 1000 px 實體純白畫布上，100% 保證匯出圖檔純淨無紅邊。
3D 材質貼圖渲染：
智慧隱藏「刀模固定圖層」，**100% 僅載入「設計稿圖層」**進行 3D 貼圖。這能徹底避免割線（紅色）、折線（啡色）在 3D 包裝表面露出，呈現完美真實的 3D 樣機效果。
2D 智慧裁切定位：
自動讀取「割線」圖層以精確辨識外圍黑色 outer 裁切線。
自動讀取「折線」圖層以精確吸附啡紅色 fold 內折線，實現 2D 網格完美貼合。
一鍵純淨導出 (1000x1000)：
在生成匯出圖檔時，系統自動將「尺寸」、「割線」、「出血」、「折線」這 4 個刀模圖層徹底剔除並關閉，只保留純淨、無紅線殘留的「設計稿圖層」，維持高畫質置中輸出。
